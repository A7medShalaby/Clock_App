<h1 align="center">Clock_App</h3>

## 🔍 Overview:

- 🐥 Used Dart & Flutter Framework with GetX state management.

- ☺ Creating Simple Ui.



## 🎯 Features:

- 🕐 Showing Current Time On Clock.

- 🔂 Switching Between 🌞Light & 🌑Dark Mode.

- 📝 Get Current Date.

- 🕛 Get Current Time.
